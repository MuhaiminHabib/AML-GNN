# AML-GNN Dissertation Project

Comparative evaluation of GCN, GraphSAGE, and GATv2 for AML node classification using the Elliptic Bitcoin dataset.

Main goals:
1. Build a shared temporal data split.
2. Train GCN, GraphSAGE, and GATv2 under identical conditions.
3. Evaluate illicit-class F1, PR-AUC, ROC-AUC, precision, recall, and confusion matrix.
4. Later evaluate explanations using faithfulness metrics.
